# R Component
rnbc →	Create a new React Native Basic Component
rnso →	Create a new React Native Stylesheet Component
rnsc →	Create a new React Native Component with Styled Component
rnscs →	Create a new Styled Component File
rnsctc →	Get the path to color theme props
rnsctf →	Get the path to fonts theme props
nbc →	Create Native Base Component

# bash
npx expo install expo-linear-gradient - gradiente
expo install @expo-google-fonts/inter expo-font - fonte
npm install @react-navigation/native - core navegaçao
npx expo install react-native-screens react-native-safe-area-context - navegaçao
npm install @react-navigation/native-stack - estrategia de navegaçao (existe tab e drawer)
npm install --save phosphor-react-native - icones
npx expo install react-native-svg - svg
npx expo install expo-clipboard - copia algo para a area de trabalho
npx expo install expo-notifications - notificação
npx expo install expo-modules-core - precisa para as notificações

npx expo start
